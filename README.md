# Certifications Portfolio Website

A modern and responsive Certifications Portfolio Website built using HTML, CSS, and JavaScript.

## 🚀 Features

- Modern UI Design
- Responsive Layout
- Category Navigation
- Skill-Based Search
- Certificate Preview
- Download Certificates
- Hover Animations
- Mobile Friendly

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```text
project-folder/
│
├── index.html
├── certificate1.png
├── certificate2.png
├── certificate3.png
```

## 🔍 Functionality

- Search certifications by:
  - Skill
  - Category
  - Certificate Name

- Filter certificates using navigation buttons.

- View and download certificates directly.
